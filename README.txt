IMAGE ACCESS / ZOOM FEATURES
----------------------------

1) Visual Comparison Section
Click image -> open enlarged image
Mouse wheel UP -> Zoom in
Mouse wheel DOWN -> Zoom out

2) Applications Section
Click image -> open enlarged image
Mouse movement -> move zoom focus
Mouse wheel -> Zoom in / out
Leaving image -> resets zoom

3) Footer Gallery
Click image -> open enlarged image
Mouse wheel -> Zoom in / out
Drag left/right to navigate
Use arrow buttons < >

4) Video
Click video icon -> opens popup player
Click video -> Play / Pause
